# OBS HTML таймер

Этот шаблон позволяет использовать один HTML-источник в OBS: фон + таймер в нужном месте.

## Как использовать

1. Откройте `index.html` в браузере и убедитесь, что таймер выглядит как нужно.
2. Замените `assets/background.svg` на свою картинку (можно PNG/JPG/SVG). Имя файла можно поменять в `index.html`.
3. В OBS добавьте источник **Browser**, включите **Local file** и укажите путь к `index.html`.
4. Поставьте размер 1920×1080 (или ваш размер сцены) и при необходимости включите `Shutdown source when not visible`.

Если превью пустое, убедитесь, что включен **Local file** и размер источника совпадает с размером сцены.

## Настройка таймера

В `script.js`:

- `DEFAULT_MINUTES` — стартовое время в минутах.
- `AUTO_START` — автозапуск.

Можно управлять таймером из консоли браузера или OBS:

```js
window.timerControls.reset(10); // 10 минут
window.timerControls.start();
window.timerControls.stop();
```

## Настройка позиции

Положение таймера задается в `styles.css` через `.scene__screen` (сдвиги `left`/`top`).

## Подсказка

Если хотите «вырезать» экран монитора, подставьте PNG с прозрачностью вместо фона — таймер останется поверх.
